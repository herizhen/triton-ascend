# Random Number Generation

| API | Brief Description |
|--|--|
|[randint4x](./randint4x.md) | Given 1 seed scalar and 1 offset block, returns 4 random blocks of type int32 |
|[randint](./randint.md) | Given 1 seed scalar and 1 offset block, returns 1 random block of type int32 |
|[rand](./rand.md)   | Given 1 seed scalar and 1 offset block, returns 1 random block of type float32 in U(0,1) |
|[randn](./randn.md)   | Given 1 seed scalar and 1 offset block, returns 1 random block of type float32 in N(0,1) |
