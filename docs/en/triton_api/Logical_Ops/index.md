# Logical Operations

| API | Brief Description |
|--|--|
| [and](./and.md) | Logical AND operation |
| [or](./or.md) | Logical OR operation |
| [not](./not.md) | Logical NOT operation |
| [logical_and](./logical_and.md) | Used to perform element-wise logical AND operation on two tensors |
| [logical_or](./logical_or.md) | Used to perform element-wise logical OR operation on two tensors |
| [not](./not.md) | Performs bitwise NOT on tensor values |
| [invert](./invert.md) | Flips each bit of every value in the tensor |
| [lshift](./lshift.md) | Performs left shift on the tensor by a given value |
| [rshift](./rshift.md) | Performs right shift on the tensor by a given value |
| [xor](./xor.md) | Computes the XOR value of two elements |
