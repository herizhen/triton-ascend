# Reduction Operations

|api|Brief Description|
|--|--|
|[argmax](./argmax.md) |Returns the indices of the maximum values along a specified axis in the input tensor |
|[argmin](./argmin.md) |Returns the indices of the minimum values along a specified axis in the input tensor |
|[max](./max.md) |Returns the maximum values along a specified axis in the input tensor |
|[min](./min.md) |Returns the minimum values along a specified axis in the input tensor |
|[reduce](./reduce.md) |Applies the combine_fn to all elements along a specified axis in the input tensor |
|[sum](./sum.md) |Returns the sum of all elements along a specified axis in the input tensor |
|[xor_sum](./xor_sum.md) |Returns the XOR sum of all elements along a specified axis in the input tensor |
